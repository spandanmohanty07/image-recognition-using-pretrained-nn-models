Objective - Classify dogs and cats using pre-trained InceptionV3 model
Environment used - Google Colab's GPU runtime environment

InceptionV3 pre-trained-model weights can be downloaded from - https://storage.googleapis.com/mledu-datasets/inception_v3_weights_tf_dim_ordering_tf_kernels_notop.h5

Cats and dogs dataset can be downloaded from - https://storage.googleapis.com/mledu-datasets/cats_and_dogs_filtered.zip
